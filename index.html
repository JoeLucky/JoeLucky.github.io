BFC

  • 一个独立的渲染区域，只有Block-level box参与
  • 它规定了内部的Block-level Box如何布局，并且与这个区域外部毫不相干。

哪些元素会生成BFC?

  • 根元素
  • float属性不为none
  • position为absolute或fixed
  • display为inline-block, table-cell, table-caption, flex, inline-flex
  • overflow不为visible

Handlebarsjs

date: 2015-07-26 16:28:39

什么是Handlebarsjs?

  • Handlebars是一个Javascript模板引擎，轻松高效的编写语义化模板
  • 简单地说只要你定义一个模版，提供一个json对象，handlebars就能吧json对象放到你所给的模版中
  • 这样能将Web前端的视图和代码分离，降低两者之间耦合

如何告诉Handlebarsjs你要用哪个模板？

只需要这么一句就可以给Handlebarsjs传递一个模板
    var source   = $("#entry-template").html();

模板可以写在HTML的\标签中，也可以写在\标签中，分别如下: 

<body>中：   <任何标签x id="entry-template">模板信息</x>  
<script>中： <script id="entry-template" type="text/x-handlebars-template">模板信息</script>

Handlebarsjs如何使用模板？

获取模板后首先需要对模板进行编译：
    var template = Handlebars.compile(source);
然后通过 var result = temolate(Data);  （其中data是一个json对象）即可获得填好数据的DOM结构
最后通过$("#").before / .html / .after(result)    即可将结果渲染到指定DOM结构前/上/后

好了，基本用法就是这样了，下面就是在 “模板信息” 中深入了

首先得知道这个“模板”是什么

Handlebars模板看起来就像是正常的Html，并使用了嵌入的 handlebars 表达式（以 双{ 开始，跟一些内容，以 双} 结束）。

这时候如果所提供的Data有如下结构： { title: “My New Post” ， … }那么返回的result结构就变成如下：

如果没有这个“填充信息”，那么双大括号以及其中内容就会置空

Handlebars的 双大括号表达式 会返回一个 HTML编码 HTML-escape 过的值 （替换特殊字符，在所有的计算机上正确读取字符）
如果不希望Handlebars来编码这些值，使用三个大括号即可：这样做的目的之一是所提供的Data对象中可以出现HTML标签信息

基础

基本符号：
    1、Markdown符号： *,-,+ （3个符号效果一样） 
    2、空白行表示另起一个段落
    3、`是表示inline代码，tab是用来标记 代码段，分别对应html的code，pre标签
换行：
    1、单一段落( <p>) 用一个空白行
    2、连续两个空格 会变成一个 <br>
    3、连续3个符号，然后是空行，表示 hr横线
特殊符号：
    1、用\来转义，表示文本中的markdown符号
    2、可以在文本种直接使用html标签，但是要注意在使用的时候，前后加上空行

字体: using # * _

标题 (using# 1~6个)

斜体 斜体 (using *word* / _word_)
加粗 加粗 (using **word** / __word__)

有||无序列表: using 数字. || */+/-

  2. something
    1. 空多4格变下一级
      1. 再空多4格变下一级
