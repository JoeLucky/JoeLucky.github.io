markdown

date: 2015-07-20

基础

基本符号：
    1、Markdown符号： *,-,+ （3个符号效果一样） 
    2、空白行表示另起一个段落
    3、`是表示inline代码，tab是用来标记 代码段，分别对应html的code，pre标签
换行：
    1、单一段落( <p>) 用一个空白行
    2、连续两个空格 会变成一个 <br>
    3、连续3个符号，然后是空行，表示 hr横线
特殊符号：
    1、用\来转义，表示文本中的markdown符号
    2、可以在文本种直接使用html标签，但是要注意在使用的时候，前后加上空行

字体: using # * _

标题 (using# 1~6个)

斜体 斜体 (using *word* / _word_)
加粗 加粗 (using **word** / __word__)

有||无序列表: using 数字. || */+/-

  1. something
    1. something
  2. something
    1. 空多4格变下一级
      1. 再空多4格变下一级
  3. something

代码引用: using >(一个空格)

> This is a header.
>> 1.This is the first list item. 
>>> ###third (can use markdown inside)        

>> 1.This is the second list item.

> another:  
> another:

链接 与 图片

链接的3种写法：
    1、[Baidu](http://www.baidu.com/ "baidu1")      
    2、<http://www.baidu.com>  
    3、[Baidu][1]  

    [1]: http://www.baidu.com  "baidu2"

效果如下：
Baidu
http://www.baidu.com/
Baidu 

图片的写法类似（第2种方法除外） 在前面加个！  (不能指定宽高 要指定只能引入html标签<img>)
1、![picture](./1.jpg "picture1")        
2、![picture][2]  

[2]: http://pic.nipic.com/2007-11-09/2007119122519868_2.jpg  "picture2"
