Handlebarsjs

date: 2015-07-26 16:28:39

什么是Handlebarsjs?

  • Handlebars是一个Javascript模板引擎，轻松高效的编写语义化模板
  • 简单地说只要你定义一个模版，提供一个json对象，handlebars就能吧json对象放到你所给的模版中
  • 这样能将Web前端的视图和代码分离，降低两者之间耦合

如何告诉Handlebarsjs你要用哪个模板？

只需要这么一句就可以给Handlebarsjs传递一个模板
    var source   = $("#entry-template").html();

模板可以写在HTML的\标签中，也可以写在\标签中，分别如下:

  • a\中：
    \<任何标签x id=”entry-template”>模板信息\
  • a\中： \

Handlebarsjs如何使用模板？

获取模板后首先需要对模板进行编译：
    var template = Handlebars.compile(source);
然后通过 var result = temolate(对象Json);  即可获得填好数据的DOM结构
最后通过$("#").before / .html / .after(result) 即可将结果渲染到指定DOM结构前/上/后
